# Contributing

Thank you for your interest in contributing to Lease Shield AI!
Before submitting a pull request, please review the following guidelines:

- Fork the repository and create your branch from `main`.
- Ensure code follows existing style and passes tests.
- Include appropriate documentation and updates.
